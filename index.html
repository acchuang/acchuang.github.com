    Octopress to Wordpress 找到一個將Octopress的文章轉移到Wordpress的方法(使用python script)

    1. 首先新增一rss.xml到source中

    layout: nil

    <?xml version=“1.0” encoding=“UTF-8”?> <atom:link rel=“hub” href=“http://pubsubhubbub.appspot.com” xmlns:atom=“http://www.w3.org/2005/Atom”/> {{ site.title | xml_escape }} Octopress {{ site.url }}

    {% for post in site.posts limit: 20 %} {{ post.title | xml_escape }} {{ post.content | expand_urls: site.url | cdata_escape }} {{ site.url }}{{ post.url }} {{ post.date | date_to_xmlschema }} {{ site.url }}{{ post.id }} {% endfor %}

    原出處:http://5minutenpause.com/blog/2012/06/08/rss-finally-working-again/[https://gist.githubusercontent.com/5minpause/2896245/raw/rss.xml] 2. Octopress使用的是atom.xml 所以要修改_config.yml中 subscribe_rss:“/rss.xml” 3. 在Terminal中執行jekyll generate然後就會將Octopress中的文章由atom.xml轉成到public/rss.xml.xml，然後再由Wordpress後台Import rss就可以了！
